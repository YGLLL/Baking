**这是一款 Android 平台上展示蛋糕制作流程的应用，用户能够选择一个食谱，并观看视频教程**

## 项目特点
- 使用Android同步框架同步数据
- 使用MediaPlayer播放网络视频 
- 适配平板 
- 使用SQLite实现数据持久化
- UI使用Loader载入数据
	
## 使用到的开源库：
- gson
- okhttp3
- litepal
- glide
